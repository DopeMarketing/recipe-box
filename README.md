# Recipe box

## Overview
A mobile and web application that allows users to digitally store, categorize, and manage their personal recipe collection. The app provides an intuitive interface for adding recipes manually or importing them from various sources, with organizational tools to help users quickly find and use their saved recipes.

## Problem
People struggle to keep track of their recipes scattered across cookbooks, websites, notes, and family handwritten cards, making it difficult to find specific recipes when needed

## Target User
Home cooks and cooking enthusiasts who want to organize their personal recipe collections digitally

## Features
- Manual recipe entry with ingredients and instructions
- Recipe categorization by cuisine, meal type, or custom tags
- Search functionality across recipe titles and ingredients
- Photo upload for recipe images
- Recipe sharing via link or export
